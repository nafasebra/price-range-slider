- Typescript

- vite
- npm

## How to use this?
- first install npm if don't have on you computer
- clone this project:
```bash
git clone https://github.com/nafasebra/price-range-slider
```
- type this command to your terminal: 
```bash
npm install
```
- for run this project, write this: 
```bash
npm run dev
```
